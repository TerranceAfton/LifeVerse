# LifeVerse

LifeVerse is a next-generation life simulation game built in Unity 6 LTS.

## Current Status

🚧 In Active Development

## Engine

- Unity 6 LTS
- Universal Render Pipeline (URP)
- C#
- Addressables
- Unity Input System

## Project Goals

- Deep life simulation
- Advanced build mode
- Persistent world simulation
- Dynamic economy
- Utility AI
- Extensive mod support
